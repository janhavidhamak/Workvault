# WorkVault (Supabase-Only)

This project runs as a Vite React app with Supabase Auth, PostgREST, and Storage.

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Set Supabase variables in `.env` or `.env.local`:
   - `VITE_SUPABASE_PROJECT_REF`
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. Run the app:
   `npm run dev`
